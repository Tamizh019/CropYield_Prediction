---

### 2. Disease Detection API

```http
POST /api/detect_disease
Content-Type: multipart/form-data
```

**Request:**
- `image`: File (JPEG/PNG of plant leaf)

**Response:**
```json
{
  "success": true,
  "disease": "Tomato_Early_Blight",
  "confidence": 0.94,
  "treatment": "Apply copper-based fungicide...",
  "prevention": "Rotate crops, remove infected leaves..."
}
```

> **Note:** Disease information is dynamically generated by Gemini AI based on the detected disease class.
